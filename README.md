This project is all about a terminal-based keystroke monitoring tool designed specifically for cybersecurity research and educational purposes. It showcases how keyboard input events can be captured and logged within a system. When you run the application, it operates in a terminal and keeps a close eye on user keystrokes. Every key you press gets recorded and saved in a log file named keylog.txt, which is stored in a designated directory. To mimic real-world keylogging behavior, the terminal hides the characters you type while still capturing the input behind the scenes. The main goal of this project is to assist security researchers and developers in understanding how keystroke logging works, examining potential security risks tied to input interception, and enhancing defensive strategies against such threats. Remember, this project is meant solely for authorized testing, cybersecurity education, and research purposes.
